#  Dashboard de Acompanhamento Financeiro | Power BI

##  Sobre o projeto

Este dashboard foi desenvolvido em **Power BI** com foco no acompanhamento financeiro de uma operação com múltiplas lojas, permitindo analisar de forma visual e estratégica indicadores como **receita**, **custos**, **despesas**, **fluxo de caixa** e **margem líquida**.

O projeto foi construído **ao vivo durante um workshop de dados promovido pela DAXUS**, empresa brasileira da área de capacitação em dados.  
O desenvolvimento foi realizado **junto com o mentor**, utilizando os materiais disponibilizados no workshop como base para a construção do dashboard.

> **Transparência importante:** este projeto **não foi criado do zero de forma totalmente autoral**, mas sim **desenvolvido de forma guiada e prática durante o workshop**, com acompanhamento em tempo real do mentor.  
> Meu objetivo com este projeto foi consolidar conhecimentos em **modelagem de dados, criação de medidas DAX, organização visual e construção de dashboards financeiros no Power BI**.

---

## 🎯 Objetivo do dashboard

O dashboard tem como objetivo oferecer uma visão consolidada da performance financeira, facilitando a análise de:

- Receita total
- Fluxo de caixa
- Margem líquida
- Evolução mensal de receitas, custos e despesas
- Despesas por categoria
- Comparativo por ano
- Desempenho financeiro por loja
- Localização das unidades no mapa

---

## 🖥️ Visão geral do painel

O painel foi estruturado para entregar uma leitura executiva rápida e intuitiva, com destaque para:

- **KPIs principais** no topo
- **Análise temporal** da evolução financeira
- **Distribuição de despesas por categoria**
- **Tabela resumo por classificação financeira**
- **Comparação do fluxo de caixa por loja**
- **Mapa com a localização das unidades**
- **Filtros por ano e loja**, permitindo navegação interativa

---

## 🧱 Estrutura do modelo de dados

O dashboard foi construído a partir de uma estrutura organizada para análise financeira, incluindo tabelas de lançamentos, lojas, calendário e medidas.

### Principais entidades utilizadas:
- **Lançamentos**  
  Base com receitas, custos e despesas

- **Lojas**  
  Informações das unidades, incluindo nome, ID e coordenadas geográficas

- **Calendário / Data**  
  Suporte para análise temporal por ano, trimestre, mês e dia

- **Tabela de medidas**  
  Organização das métricas DAX utilizadas no projeto

Este projeto foi desenvolvido com base em **materiais disponibilizados pela DAXUS** durante um **workshop de dados**, em um formato prático e aplicado.
